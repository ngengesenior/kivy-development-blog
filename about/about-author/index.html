    About me

    I am Ngenge Senior and a Bsc. student of the University of Buea, Cameroon.I have been studying Python for close to two and a half years now and Kivy for a about one and a half years. I am also part of Njaka which is aimed at introducing kids to programming through Scratch and web development.

    How I got to know Kivy

    All the codes that I wrote initially when I started Python programming in Python was always code that ran in the console.But I got inspired by a friend of mind when he built a calculator app using Tkinter which is part of Python itself.I gave a try to Tkinter but it was a bit boring to me so I had to do an internet search on best GUI frameworks for Python. That is how I came to know Kivy. Today, I try to render 70% of my test apps using Kivy.
